## When to use

Use this skill when you are:
- Reading EigenLayer strategy metadata and restaking positions
- Inspecting delegation state, queued withdrawals, or reward metadata
- Depositing into EigenCloud strategies or changing delegation
- Completing queued withdrawals or claiming rewards from prepared claim data
